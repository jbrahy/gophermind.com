# GopherLogo.md

## ASCII Art Banner

A Midjourney-generated "genius gopher" image has been converted to ASCII art and embedded in the codebase.

### Source

- **Original image**: a Midjourney-generated "genius gopher" illustration (local source, not committed)
- **Format**: PNG, 1024×1024, 8-bit/color RGB
- **Conversion**: Python + Pillow, grayscale resize to 100×45, mapped to `@%#*+=-:. ` character ramp

### Code Location

- **File**: `internal/prompt/art.go`
- **Constant**: `GopherArt` (unexported, package-private)
- **Package**: `prompt`

### Usage

The ASCII art is embedded as a Go string constant. It can be displayed at startup by printing `prompt.GopherArt` from the TUI or CLI entry point.

### Conversion Details

| Parameter | Value |
|-----------|-------|
| Source width | 100 chars |
| Source height | ~45 lines |
| Character ramp | `@%#*+=-:. ` (dark → light) |
| Aspect ratio correction | 0.45 (terminal chars are ~2× taller than wide) |
| Grayscale | L mode (Pillow) |
